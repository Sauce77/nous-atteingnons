# nous-atteingnons
Buscador de alcance total de personas en una organizacion.

-------------------------------------------------
  GUÍA RÁPIDA PARA EJECUTAR LA APLICACIÓN
--------------------------------------------------

Antes de empezar, asegúrate de tener Python instalado.

1.  **Instalar Python:**
    - Ve a: https://www.python.org/downloads/
    - Descarga la última versión de Python.
    - **¡MUY IMPORTANTE!** En el instalador, asegúrate de marcar la casilla que dice "Add Python to PATH".



Una vez instalado Python, simplemente haz doble clic en el archivo "iniciar.bat" (en Windows) o ejecuta "./iniciar.sh" desde una terminal (en macOS/Linux).

¡Eso es todo! La primera vez tardará un poco mientras instala las dependencias.
